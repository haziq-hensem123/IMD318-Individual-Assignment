<section id="bio" data-aos="fade-up" data-aos-duration="1500">
<h1 class="section-title">Biography</h1>
<p>Born on 5 April 1972 in Sarawak, Bernard Radin—also known as "The Head Hunter"—stands as one of Malaysia’s most decorated multi-disciplinary martial artists. Holding Iban and German heritage, he grew up in a longhouse in Roban, Sarikei, surrounded by ancestral warrior traditions. His fighting name reflects the legendary Iban warrior customs of headhunting, symbolising bravery and fearlessness.</p>
<p>Standing at 5'7" and competing at 70kg, he built a fierce reputation in Muay Thai, Taekwondo, and Chinese Kickboxing. With a record of 45 fights, 40 wins, and 32 knockouts, Bernard earned a place among Malaysia’s greatest fighters.</p>

  // TIME AND DATE SCRIPT
  function updateClock() {
    const now = new Date();
    
    // Time Formatting
    const timeString = now.toLocaleTimeString('en-US', { 
        hour12: false, 
        hour: '2-digit', 
        minute: '2-digit', 
        second: '2-digit' 
    });
    
    // Date Formatting
    const dateString = now.toLocaleDateString('en-GB', { 
        weekday: 'long', 
        year: 'numeric', 
        month: 'long', 
        day: 'numeric' 
    });

    document.getElementById('current-time').textContent = timeString;
    document.getElementById('current-date').textContent = dateString.toUpperCase();
  }

  // Update immediately then every second
  updateClock();
  setInterval(updateClock, 1000);
